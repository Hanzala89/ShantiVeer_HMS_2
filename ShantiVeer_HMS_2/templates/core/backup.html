Auto-Backup Setup

Automatic backups need a daily scheduler to call the management command:

python manage.py run_scheduled_backup
Linux cron (runs daily at 2 AM):
0 2 * * * /venv/bin/python /path/manage.py run_scheduled_backup
Windows Task Scheduler:
Set to run daily, action = python manage.py run_scheduled_backup
